This repository contains all the files for the assesment 3: multiple protein sequence alignment. 


In the "GRAPHS" folders you can find the graphs for the different sets.

The R_Plotting contains the script for R used to egenerate the plots.

The parse_alig.md contains the script used to extract the data from the output files.

The "Sequences folder" contains all the files used to analyse the data related with the protein sequences I downoloaded from UniProt, and also all the 
output files generated during the process, as well as the graohs relkated with this set of data.
